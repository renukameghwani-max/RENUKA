# RENUKA - Professional Landing Page

A modern, responsive landing page built with clean HTML, CSS, and JavaScript. Perfect for showcasing your product or service with a professional design.

## 🎨 Features

### Navigation Bar
- Sticky navigation with logo branding
- Menu links: Features, Pricing, Testimonials, Contact
- "Get Started" CTA button
- Responsive mobile menu support

### Hero Section
- Bold, engaging headline with gradient text
- Compelling subheadline explaining value proposition
- Dual CTA buttons (Primary & Secondary)
- Abstract SVG illustration that floats smoothly
- Fully responsive grid layout

### Social Proof Bar
- Dark background design element
- Key metrics: Active Users, Satisfaction Rate, Support, Enterprise Clients
- Animated counter effects
- Dividers between stats

### Features Section
- Grid layout showcasing 3 key features
- Hover effects with smooth animations
- Icons and descriptions
- Responsive card design

### Call-to-Action Section
- Gradient background matching brand colors
- Eye-catching headline
- Large action button
- Encourages final conversions

### Footer
- Simple, clean footer
- Copyright information

## 🚀 Getting Started

1. **Clone or download the repository**
   ```bash
   git clone https://github.com/renukameghwani-max/RENUKA.git
   cd RENUKA
   ```

2. **Open in your browser**
   - Simply open `index.html` in your web browser
   - Or serve with a local development server:
   ```bash
   python -m http.server 8000
   # or with Node.js
   npx http-server
   ```

3. **Customize for your brand**
   - Edit the text in `index.html`
   - Update colors in `styles.css` (`:root` variables)
   - Modify animations and interactions in `script.js`

## 📱 Responsive Design

The page is fully responsive with breakpoints for:
- Mobile devices (< 768px)
- Tablets (768px - 1024px)
- Desktop (> 1024px)

## 🛠️ Customization Guide

### Change Brand Color
Edit the CSS variables in `styles.css`:
```css
:root {
    --primary-color: #667eea;      /* Change this */
    --secondary-color: #764ba2;    /* Change this */
    --accent-color: #f093fb;       /* Change this */
}
```

### Update Content
Replace placeholder text in `index.html`:
- Logo text: "RENUKA"
- Headlines and subheadlines
- Button text
- Feature descriptions
- Social proof statistics

### Modify SVG Illustration
Edit the SVG in the hero section to match your brand or replace with your own image.
